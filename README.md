# Community (Vue 3 Boilerplate)

Vue 3.0 starter project for Maplr Community
Using Typescript

## Setup

```
yarn install
yarn serve
```

## Start Fake Backend

```
yarn start-fakebackend
```

## Launch Test

```
yarn test:unit
yarn test:e2e
```

### Customize configuration

See [Configuration Reference](https://cli.vuejs.org/config/).
